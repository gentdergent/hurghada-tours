## Projekt-Kurzbeschreibung
Webseite für ein Reisebüro in Hurghada am Roten Meer. Mittelsmann-Geschäftsmodell — vermittelt Trips an lokale Anbieter.

**Angebot:**
- Tauchen / Diving Courses
- Submarine-Trips
- Luxor Day-Trip + Overnight
- Cairo Day-Trip + Overnight
- Desert Safaris (Quad/Jeep)
- Boat Trips (Orange Bay, Paradise Island, Dolphin House)

**Zielgruppe:** Europäische Touristen vor Ort in Hurghada (DE/EN primär, RU/PL später)

### Gemeinsame Muster (Must-Haves)

**Booking-Flow (WICHTIGSTER INSIGHT):**
- Online-Payment ist KEIN Standard in der Branche
- WhatsApp ist Haupt-Kanal (reduziert Friction, matcht int. Publikum)
- "Build Your Trip"-Formular → Email + WhatsApp
- Preise sichtbar, Buchung läuft konversationsbasiert
- **Konsequenz:** V1 ohne Payment-Gateway, nur WhatsApp + Formular

**Navigation:**
- Achse 1: Location (Hurghada, El Gouna, Makadi, Soma, Sahl Hasheesh)
- Achse 2: Aktivitätstyp (Sea, Cultural, Desert, Multi-Day)

**Sprachen:** DE + EN minimum. RU/PL für ROI-Expansion.

**Trust-Signale:**
- TripAdvisor / HolidayCheck Badges
- "X Jahre Erfahrung", "Y+ zufriedene Kunden"
- 24/7 Support-Indicator
- Price Match Guarantee
- Echte Kundenfotos + Reviews

**Preis-Benchmarks (Markt-Orientierung):**
- Boat/Snorkel: €25–75
- Day-Trip Luxor/Cairo: €150–260
- Multi-Day (Luxor overnight + Cairo): €300–450
- Nile Cruise (4 Tage): €450–875

**Design-Sprache:**
- Gold/Türkis + Weiß (Egypt-Touch ohne Kitsch)
- Große Hero-Bilder (Rotes Meer, Pyramiden, Wüste)
- Grid-Galleries 3er-Reihen
- Transparenter Navbar → solid on scroll
- Hover-Animationen, smooth transitions
- Mobile-First (Touristen nutzen Handy im Resort)

## Site-Struktur V1

```
/                       Hero + Top 6 Tours + Trust Signals + CTA + WhatsApp
/tours                  Alle Touren, filterbar
/tours/diving           Tauchen + Submarine
/tours/sea              Boat Trips, Orange Bay, Dolphin House
/tours/cultural         Luxor, Cairo (Day + Overnight)
/tours/desert           Safaris (Quad, Jeep, Stargazing)
/tours/[slug]           Einzelne Tour — Gallery, Details, FAQ, CTA
/about                  Über uns + Team + Lizenzen
/contact                WhatsApp + Formular + Map
```

**Tour-Detail-Seite Komponenten:**
- Hero-Slider 5–8 Bilder
- Sidebar (sticky): Dauer, Preis, Inklusive, Treffpunkt, Sprachen
- Description-Tabs: Overview, Itinerary, Inklusive, FAQ
- Ähnliche Touren
- WhatsApp-CTA fixed bottom bar (mobile)
- "Build Your Trip"-Formular

## Tech-Stack

| Layer | Tool | Grund |
|-------|------|-------|
| Framework | Next.js 15 (App Router) | SEO, i18n, Image-Optimierung |
| Language | TypeScript | Type-Safety |
| Styling | Tailwind CSS v4 | Speed, Consistency |
| Components | shadcn/ui | Qualitativ, anpassbar |
| i18n | next-intl | App Router kompatibel |
| Animation | Framer Motion | Smooth Transitions |
| CMS | Sanity.io | Kunde/Betreiber kann Touren selbst pflegen |
| Forms | React Hook Form + Zod | Validation |
| Email | Resend | Einfach, günstig |
| Analytics | Plausible | DSGVO ohne Cookie-Banner |
| Hosting | Vercel | DE-Edge, schnell für EU |
| SEO | next-sitemap + Schema.org | TravelAgency, Offer Markup |
